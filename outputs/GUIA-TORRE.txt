INTEGRACAO SCANIA/rFMS

Objetivo
--------
A fonte antiga da Torre foi substituida pela API oficial Scania Data Access
rFMS. O dashboard continua chamando a rota interna /api/torre para nao exigir
mudancas na tela, mas por tras a funcao consulta diretamente:

- /vehicles
- /vehiclepositions
- /vehiclestatuses

O que a funcao entrega ao dashboard
-----------------------------------
- Total de veiculos monitorados
- Veiculos rodando e parados
- KM hoje por odometro Scania
- Litros e KM/L por diferenca de acumuladores
- Nivel de diesel
- Nivel de ARLA
- Velocidade atual
- Velocidade maxima em 24h
- Percentual acima de 100 km/h
- Alertas por luzes RED, YELLOW ou AMBER

Como configurar no Netlify
--------------------------
1. Abra o projeto no Netlify.
2. Entre em Project configuration > Environment variables.
3. Mantenha a variavel TAZ_ADMIN_KEY.
4. Crie a variavel SCANIA_RFMS_CONFIG_JSON.
5. Cole nela o conteudo do seu frota_config.json.
6. Faca um novo deploy.

O formato esperado e:

{
  "empresas": [
    {
      "empresa": "TAZ",
      "ativo": true,
      "base_url": "https://dataaccess.scania.com/rfms4",
      "client_id": "SEU_CLIENT_ID",
      "client_secret": "SEU_CLIENT_SECRET"
    }
  ]
}

Alternativa por variaveis separadas
-----------------------------------
Se preferir nao colar um JSON grande, use estas variaveis:

SCANIA_TAZ_CLIENT_ID
SCANIA_TAZ_CLIENT_SECRET
SCANIA_PRATAZIL_CLIENT_ID
SCANIA_PRATAZIL_CLIENT_SECRET
SCANIA_RILUCA_CLIENT_ID
SCANIA_RILUCA_CLIENT_SECRET

As variaveis *_BASE_URL sao opcionais. Se nao forem criadas, a funcao usa:
https://dataaccess.scania.com/rfms4

Variaveis opcionais
-------------------
SCANIA_CACHE_SECONDS
- Padrao: 120.
- Controla por quantos segundos a resposta da Scania fica em cache no servidor.
- Ajuda a evitar limite 429 da Scania quando mais de uma pessoa abre o dashboard.

SCANIA_HISTORY_MAX_PAGES
- Padrao: 6.
- Controla o maximo de paginas historicas por empresa para calcular KM hoje e
  picos de velocidade.

Como consultar no dashboard
---------------------------
1. Abra:
   https://tazmotoristas.netlify.app/dashboard_taz_integrado.html
2. Entre com e-mail e senha do Supabase.
3. O dashboard carrega a Base online e a Scania/rFMS ao mesmo tempo.
4. O botao Atualizar online pode ser usado para recarregar manualmente.
5. Depois do primeiro carregamento, a consulta roda automaticamente a cada
   2 minutos enquanto o dashboard estiver aberto.
6. Usuarios role=empresa visualizam somente as empresas listadas em companies.

Seguranca
---------
Nao coloque client_id, client_secret ou frota_config.json em nenhum arquivo do
projeto. Esses dados devem ficar somente nas variaveis do Netlify.

O arquivo .gitignore ja bloqueia frota_config.json para evitar envio acidental
ao GitHub.

Se os segredos foram compartilhados fora do ambiente privado, o ideal e gerar
novos pares client_id/client_secret no portal da Scania.
