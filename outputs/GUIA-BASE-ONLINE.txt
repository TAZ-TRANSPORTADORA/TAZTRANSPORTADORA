BASE CENTRAL ONLINE DO APP TAZ

ARQUITETURA
Celulares -> Funcoes Netlify -> Netlify Blobs -> Excel consolidado

O Netlify Blobs passa a ser a base principal. Cada viagem e gravada separadamente
pelo seu ID. Isso permite reunir os dados de todos os aparelhos e evita duplicar
um registro quando ele for reenviado.

PUBLICACAO
Publique o projeto completo pelo GitHub conectado ao Netlify. Estes itens devem
estar no repositorio:
- package.json
- netlify.toml
- netlify/functions
- outputs

O Netlify instalara automaticamente a dependencia @netlify/blobs durante o build.

CONFIGURACAO OBRIGATORIA
No Netlify:
1. Abra Project configuration > Environment variables.
2. Crie TAZ_ADMIN_KEY.
3. Defina um codigo forte, conhecido apenas pela administracao.
4. Faca um novo deploy.

Nao coloque esse codigo em nenhum arquivo do projeto.

COMO FUNCIONA
- O motorista toca em Salvar registro.
- O aparelho guarda uma copia local.
- Com internet, a viagem e enviada para a base central.
- Sem internet, o registro fica pendente e tenta novamente.
- Reenviar a mesma viagem atualiza o registro de mesmo ID, sem duplicacao.
- Motoristas, cavalos e carretas tambem ficam na base central.

BAIXAR TODOS OS DADOS
1. Abra o app.
2. Na secao Base online, digite o codigo TAZ_ADMIN_KEY.
3. Toque em Baixar base online.
4. O app gera Base_TAZ_online_DATA.xlsx contendo:
   - APP_VIAGENS
   - APP_ABASTECIMENTOS
   - CAD_MOTORISTAS
   - CAD_CAVALOS
   - CAD_CARRETAS

USAR NO DASHBOARD
1. Abra dashboard_taz_integrado.html no site.
2. Em Atualizar dados, digite o mesmo TAZ_ADMIN_KEY.
3. Clique em Base online.
4. As viagens e abastecimentos de todos os aparelhos serao somados aos dados
   historicos do dashboard.

POWER AUTOMATE
POWER_AUTOMATE_URL agora e opcional. Se estiver configurada, cada viagem tambem
sera espelhada no Excel do OneDrive. Se nao estiver, a gravacao na base central
continua funcionando normalmente.

ROTAS
- POST /api/sync-excel: grava viagens.
- POST /api/sync-cadastros: grava ou consulta cadastros.
- POST /api/base-online: consulta administrativa protegida.

SEGURANCA
A base completa so pode ser baixada com TAZ_ADMIN_KEY. Os motoristas nao precisam
conhecer esse codigo para registrar viagens.
