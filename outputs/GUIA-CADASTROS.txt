CADASTROS COMPARTILHADOS: MOTORISTAS, CAVALOS E CARRETAS

O app ja possui a tela Cadastros e funciona localmente. Para compartilhar as
listas entre todos os celulares, configure este segundo fluxo.

1. PLANILHA
Use a versao atualizada de Consolidado_TAZ_integrado.xlsx.

Transforme cada intervalo em uma Tabela do Excel:
- CAD_MOTORISTAS -> tbMotoristas
- CAD_CAVALOS -> tbCavalos
- CAD_CARRETAS -> tbCarretas

Colunas:
- Motoristas: ID, Nome, Empresa, Categoria, Ativo, Atualizado em
- Cavalos e carretas: ID, Placa, Ativo, Atualizado em

2. POWER AUTOMATE
Crie um fluxo com "Quando uma solicitacao HTTP for recebida".

O fluxo recebe:
{
  "operation": "list ou upsert",
  "type": "drivers, horses ou trailers",
  "item": {
    "id": "identificador",
    "name": "nome ou placa",
    "company": "empresa",
    "category": "categoria",
    "active": true,
    "updatedAt": "data ISO"
  }
}

Para operation = upsert:
- Use uma Condicao ou Switch para escolher a tabela pelo campo type.
- Liste as linhas da tabela usando o filtro ID eq 'item.id'.
- Se encontrar, use "Atualizar uma linha".
- Se nao encontrar, use "Adicionar uma linha".
- Para drivers, Nome recebe item.name.
- Para horses/trailers, Placa recebe item.name.
- Ativo recebe item.active.
- Atualizado em recebe item.updatedAt.
- Finalize com uma acao Resposta, status 200 e corpo {"saved": true}.

Para operation = list:
- Liste as linhas de tbMotoristas, tbCavalos e tbCarretas.
- Use a acao "Selecionar" para converter as colunas para o formato abaixo.
- Finalize com uma acao Resposta, status 200:
{
  "registries": {
    "drivers": [
      {
        "id": "ID",
        "name": "Nome",
        "company": "Empresa",
        "category": "Categoria",
        "active": true,
        "updatedAt": "Atualizado em",
        "syncStatus": "synced"
      }
    ],
    "horses": [
      {
        "id": "ID",
        "name": "Placa",
        "company": "",
        "category": "",
        "active": true,
        "updatedAt": "Atualizado em",
        "syncStatus": "synced"
      }
    ],
    "trailers": [
      {
        "id": "ID",
        "name": "Placa",
        "company": "",
        "category": "",
        "active": true,
        "updatedAt": "Atualizado em",
        "syncStatus": "synced"
      }
    ]
  }
}

3. NETLIFY
- Copie a URL HTTP do novo fluxo.
- No Netlify, crie a variavel POWER_AUTOMATE_CADASTROS_URL.
- Publique novamente o projeto completo.

Rota usada pelo app:
https://taztransportadora.netlify.app/api/sync-cadastros

Teste:
Abrir essa rota no navegador deve retornar "Metodo nao permitido". Isso confirma
que a funcao existe. O cadastro real e enviado pelo app usando POST.

COMO USAR NO APP
1. Abra https://taztransportadora.netlify.app/
2. Toque na aba Cadastros.
3. Escolha Motoristas, Cavalos ou Carretas.
4. Preencha e toque em Salvar cadastro.
5. Use Editar para corrigir e Desativar para retirar das sugestoes.
6. Toque em Atualizar para buscar os cadastros compartilhados no Excel.
